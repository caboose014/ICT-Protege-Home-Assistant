# ICT Protege Automation for Home Assistant

</p>

A custom integration to control ICT Protege WX/GX systems via Home Assistant.

## Features
* **Doors:** Lock/Unlock and Open/Closed status.
* **Areas:** Arm (Away/Stay/Night) and Disarm.
* **Inputs:** View status and Bypass/Unbypass via dropdowns.
* **Outputs:** Control PGMs and other outputs.
* **Troubles:** Monitor system trouble inputs (Battery, Tamper, etc.).

## Installation
1. Click "Install" in HACS.
2. Restart Home Assistant.
3. Go to Settings > Devices & Services > Add Integration.
4. Search for "ICT Protege Automation".
